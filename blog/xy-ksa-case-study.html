SEO Case Study: xy-ksa.com

A complete technical SEO audit of a Korean beauty products store on the Salla e-commerce platform in Saudi Arabia.

Published: July 23, 2026 Category: Technical SEO Platform: Salla

Overview

xy-ksa.com is a Saudi e-commerce store selling Korean beauty and skincare products. The site is built on the Salla platform, which is the leading Arabic e-commerce platform in the Middle East. We conducted a comprehensive SEO audit using our Python automation tools to identify technical, content, and performance issues.

Technical SEO Findings

Issues Found

Issue Severity Impact
Multiple H1 tags on homepage Critical Confuses search engines about page topic
Thin content (188 words on homepage) Critical Below 300-word minimum for ranking
No gzip compression Warning Slower page load times
No Cache-Control headers Warning Repeated downloads on return visits
No lazy loading on images Warning Initial page load downloads all images
Missing Open Graph tags Warning Poor social media sharing preview
Zero backlinks Critical No domain authority building

Content SEO Findings

  • Word Count: 188 words (too thin for ranking)
  • H1 Tags: Multiple H1 tags - should be exactly one
  • Image Alt Tags: 100% coverage (good)
  • Internal Links: Good internal linking structure
  • E-E-A-T Score: 0/100 (no author, no expertise signals)

Speed Analysis

The site scored 42/100 on speed. Key issues:

  • No gzip compression enabled
  • No browser caching headers
  • Images not optimized for web
  • No lazy loading implementation
  • Heavy JavaScript payload from Salla platform

Recommendations

Quick Wins (Do First)

  • Fix H1 tags - ensure only one H1 per page
  • Add more content to homepage (aim for 300+ words)
  • Enable gzip compression in Salla settings
  • Add Cache-Control headers

Medium-Term (1-3 Months)

  • Build backlinks through outreach and content marketing
  • Add Open Graph tags for social sharing
  • Create a blog with Korean beauty content
  • Implement lazy loading for images

Long-Term (3-6 Months)

  • Build domain authority through consistent link building
  • Optimize for AI overviews and LLM search
  • Create comprehensive product category pages
  • Implement schema markup for products
